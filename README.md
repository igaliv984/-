# Каталог керамогранитной плитки

Современный сайт-каталог для магазина керамогранитной плитки с интеграцией Google Sheets.

## 🚀 Быстрый старт

### 1. Подключение Google Sheets

1. Создайте новую Google Таблицу
2. Добавьте заголовки в первую строку:
   ```
   name, sku, size, surface, purpose, color, price_m2, stock, image_url, description
   ```
3. Заполните данными (минимум 5-10 товаров для теста)
4. **Файл → Опубликовать в интернете**
5. Выберите "Весь документ" и формат "CSV"
6. Скопируйте полученную ссылку
7. Вставьте ссылку в переменную `SHEET_URL` в файле `index.html` (строка ~280)

### 2. Настройка формы заявок

1. Зарегистрируйтесь на [Formspree](https://formspree.io/)
2. Создайте новую форму
3. Скопируйте ваш Form ID
4. Замените `YOUR_FORM_ID` в файле `index.html` (строка ~520)

### 3. Деплой на GitHub Pages

```bash
# 1. Инициализируйте репозиторий (если ещё не создан)
git init
git add .
git commit -m "Initial commit"

# 2. Создайте репозиторий на GitHub
# 3. Привяжите remote
git remote add origin https://github.com/your-username/your-repo.git
git push -u origin main

# 4. В настройках репозитория: Settings → Pages → Source → main branch
# 5. Сайт будет доступен по адресу: https://your-username.github.io/your-repo/
```

### 4. Деплой на Netlify

1. Зарегистрируйтесь на [Netlify](https://netlify.com)
2. Перетащите папку с проектом в окно браузера (Drag & Drop)
3. Готово! Сайт опубликован

### 5. Структура данных Google Sheets

| Column | Description | Example |
|--------|-------------|---------|
| name | Название коллекции | "Atlas Concorde Boost" |
| sku | Артикул | "AC-BOOST-60" |
| size | Формат | "60x60", "120x60" |
| surface | Поверхность | "матовая", "полированная", "структурированная" |
| purpose | Назначение | "пол", "стена", "фасад" |
| color | Цвет/оттенок | "серый", "бежевый", "белый" |
| price_m2 | Цена за м² | "2500" |
| stock | Наличие | "в наличии", "под заказ", "нет в наличии" |
| image_url | Ссылка на фото | "https://..." |
| description | Описание | "Керамогранит для пола..." |

## ✨ Особенности

- **SPA архитектура** — быстрая навигация без перезагрузки
- **Живые фильтры** — мгновенная фильтрация без перезагрузки
- **Калькулятор плитки** — расчёт количества и стоимости
- **Адаптивный дизайн** — корректно работает на всех устройствах
- **SEO-оптимизация** — базовые мета-теги для поисковиков
- **Lazy loading** — оптимизированная загрузка изображений
